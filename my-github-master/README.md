## ğŸš€ How to Host on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in to your account
2. Click the **+** icon and select **New repository**
3. Name it (e.g., `portfolio` or `web-projects`)
4. Add a description (optional)
5. Choose **Public** so it's accessible
6. Click **Create repository**

### Step 2: Push Your Code to GitHub
```bash
# Initialize git in your local folder
git init

# Add all files
git add .

# Commit the changes
git commit -m "Initial commit: Add portfolio projects"

# Add remote repository
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings**
3. Scroll to **Pages** section
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**

### Step 4: Access Your Website
Your site will be published at: `https://YOUR_USERNAME.github.io/REPO_NAME/`
